# Ontology-Based Knowledge Graph System

A multi-agent AI system for dynamic business knowledge management using Neo4j and Gemini AI.

## Features

- **Base Business Ontology**: Generic model adaptable to any industry
- **Natural Language Queries**: Ask questions in plain English
- **Multi-Agent System**: Proposer → Validator → Critic pipeline
- **Dynamic Knowledge Graph**: Evolves based on user observations
- **Real-time Visualization**: See your knowledge graph grow

## Prerequisites

- Node.js 18+
- Neo4j Desktop or Neo4j AuraDB account
- Google Gemini API key
